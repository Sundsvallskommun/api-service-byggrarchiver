När ett dokument skulle arkiveras i LTA uppstod ett problem.
Filtypen på dokumentet stöds inte. Konvertera dokumentet till något av de tillåtna filformaten:

  • .pdf
  • .docx
  • .doc
  • .xlsx
  • .xls
  • .pptx
  • .tif
  • .rtf
  • .msg
  • .png
  • .jpeg

Uppgifter om dokumentet:

  • Ligger i ärende: ${byggrCaseId}
  • Dokumentnamn: ${documentName}
  • Handlingstyp: ${documentType}

OBS! Om dokumentet är konverterat till rätt format vid nästa körning kommer arkiveringen att slutföras korrekt.
